# Nexus: React


## Project Structure
Your project should look something like this:

```
root
+-- node_modules
+-- src
    +-- process
        +-- main.ts

    +-- renderer        // unless you are making an internal module
        +-- renderer.ts
        +-- styles.css
        

    +-- electron.ts 
    +-- export-config.js
    +-- module-info.json

+-- .gitattributes
+-- .gitignore
+-- package-lock.json
+-- package.json
+-- README.md
+-- tsconfig.json

```
